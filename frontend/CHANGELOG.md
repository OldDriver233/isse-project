# 更新日志

## v1.1.0 - 2025-11-16

### ✨ 新功能

- **多大师支持**
  - 添加大师选择器，支持选择不同的社会学大师
  - 当前支持托克维尔，未来将添加更多大师
  - 每位大师的对话独立存储和管理

- **对话保留机制**
  - 页面切换后对话内容自动保留
  - 刷新浏览器后对话自动恢复
  - 按大师分组保存对话历史

### 🔄 改进

- **UI优化**
  - 移除历史页面，简化界面结构
  - 移除语言切换和界面密度设置（未实现功能）
  - 优化导航栏布局
  - 改进大师头像显示

- **设置页面**
  - 添加清除所有对话功能
  - 添加清除反馈数据功能
  - 移除未实现的功能选项

### 🗑️ 移除功能

- 移除历史记录页面
- 移除语言切换功能
- 移除界面密度设置
- 移除字数提示
- 移除引用功能

### 📝 文档更新

- 更新 UI_DESIGN.md 设计规范
- 更新 README.md 功能说明
- 更新 QUICKSTART.md 使用指南
- 添加 CHANGELOG.md 更新日志

### 🐛 Bug修复

- 修复页面切换后对话丢失的问题
- 修复大师切换时对话混淆的问题
- 优化响应式布局在移动端的显示
