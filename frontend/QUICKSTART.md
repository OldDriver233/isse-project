# 快速开始指南

## 前提条件

1. ✅ 后端服务已启动（端口 8000）
2. ✅ Python 3.x 已安装

## 启动步骤

### Windows 用户

双击运行 `start.bat`

或在命令行中：
```bash
cd frontend
start.bat
```

### Linux/Mac 用户

```bash
cd frontend
chmod +x start.sh
./start.sh
```

### 手动启动

```bash
cd frontend
python -m http.server 3000
```

## 访问应用

1. **主应用**: http://localhost:3000
2. **API测试**: http://localhost:3000/test.html
3. **后端文档**: http://localhost:8000/docs

## 使用说明

### 1. 选择大师

- 在聊天页面顶部的下拉菜单中选择大师
- 当前支持：托克维尔
- 更多大师即将加入

### 2. 开始对话

- 选择大师后，在输入框输入问题
- 例如："托克维尔如何看待民主？"
- 按 Enter 发送（Shift+Enter 换行）
- 等待 AI 回复

### 3. 切换大师

- 点击大师选择器切换到其他大师
- 每位大师的对话独立保存
- 切换后之前的对话不会丢失

### 4. 查看对话

- 对话会自动保存在浏览器中
- 页面切换后对话保留
- 刷新浏览器后对话恢复
- 可在设置中清空所有对话

### 5. 提交反馈

- 在对话页面，点击消息下方的"反馈"按钮
- 选择评分（1-10）
- 可选填写评语
- 点击"提交"

### 6. 查看统计

- 点击"统计"查看使用数据
- 包括平均评分、反馈数等

### 7. 设置

- 点击"设置"
- 可切换深色/浅色主题
- 可清除所有对话或反馈数据

## 故障排查

### 问题：消息发送失败

**解决方案：**

1. 检查后端是否运行：
   ```bash
   curl http://localhost:8000/health
   ```

2. 使用测试页面诊断：
   - 访问 http://localhost:3000/test.html
   - 点击"测试健康检查"
   - 点击"测试聊天 API"

3. 检查浏览器控制台（F12）是否有错误

### 问题：无法访问前端

**解决方案：**

1. 确认端口 3000 未被占用
2. 检查 Python 是否正确安装
3. 尝试更换端口：
   ```bash
   python -m http.server 3001
   ```

### 问题：CORS 错误

**解决方案：**

检查后端配置文件 `backend/app/config.py`：
```python
ALLOWED_ORIGINS = ["http://localhost:3000", ...]
```

确保包含您使用的前端地址。
